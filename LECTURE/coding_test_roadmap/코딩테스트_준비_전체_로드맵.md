# 코딩테스트 준비 전체 로드맵

목표:
- 대기업 / 공기업 코딩테스트 **실전 통과**
- 문제 유형 판단 → 알고리즘 선택 → 구현까지 **자동화**
- “본 적 없는 문제”에서도 흔들리지 않는 사고력 확보

원칙:
- 백준 / solved.ac는 **학습자료**
- 커리큘럼의 핵심은 **사고 훈련과 판단 공식**
- 문제 수 < 문제 분석 깊이

---

## 전체 구조 개요

Phase 0. 전투 준비 (기초 고정)
Phase 1. 실버 안정화 (판단력 형성)
Phase 2. 실버 → 골드 분기 돌파
Phase 3. 골드 실전 완성
Phase 4. 기업 코딩테스트 실전 대응
Phase 5. 최종 점검 & 약점 제거

---

## Phase 0. 전투 준비 (1주)

목표:
- 환경, 템플릿, 사고 흐름 완전 고정
- “문제 풀 준비 상태” 만들기

내용:
- Python 코딩테스트 표준 템플릿 고정
- 빠른 입출력, 기본 자료구조 사용 습관
- 문제 풀이 5단계 사고 루틴 확립

학습 자료:
- 백준 단계별로 풀어보기 (선별)
  - 입출력
  - 배열
  - 문자열
  - 정렬

과외 포인트:
- 개념 설명 최소화
- 실수 패턴 제거

통과 기준:
- 실버 문제를 겁 없이 시작할 수 있음

---

## Phase 1. 실버 안정화 (Class 2 → Class 3) (3~4주)

목표:
- 실버 문제를 유형으로 즉시 분류
- 시간 초과/메모리 초과 사고 제거

핵심 훈련:
- 스택 / 큐 / 덱 / 힙 판단
- 완전탐색 vs 가지치기

학습 자료:
- solved.ac Class 2 전체
- solved.ac Class 3 (선별)

통과 기준:
- 문제 유형을 30초 내 판단 가능

---

## Phase 2. 실버 → 골드 분기 돌파 (Class 3 후반 ~ Class 4) (4~5주)

목표:
- 알고리즘 선택 실패 최소화

핵심 훈련:
- DFS vs BFS
- DP vs 그리디
- 투 포인터 / 이분 탐색

---

## Phase 3. 골드 실전 완성 (Class 4 → Class 5) (4~6주)

목표:
- 골드 문제 완주 능력 확보

핵심 훈련:
- 고급 DP
- 그래프 알고리즘
- 복합 시뮬레이션

---

## Phase 4. 기업 코딩테스트 실전 대응 (2~3주)

목표:
- 시험 전략 완성

핵심 훈련:
- 문제 선택 전략
- 시간 관리

---

## Phase 5. 최종 점검 & 약점 제거

목표:
- 개인 약점 제거

---

## 최종 도착점

- 문제 접근이 두렵지 않음
- 합격선 계산 가능
